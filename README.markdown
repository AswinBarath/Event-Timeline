## What's unique in this Project

- The Event Timeline uses CSS Animataions to dynamically change the background and pop up each event when we scroll.
- The Event Timeline project uses Pug JS template engine for HTML5 pre-processing, making the HTML code more readable.
- SASS preprocessor is used as scripting language that compiles into Cascading Style Sheets, making the CSS style more readable.
- The project is deployed using Node JS and Express JS file server on top of Repl it platform.
